## Profile

Chris Kenst is a software testing and developer community leader with two decades of experience across quality engineering, test automation, education, and technical advocacy.

This portfolio presents public examples of developer relations work: writing, talks, code samples, community programs, and product feedback loops. It is adapted from private portfolio material into a web format that is easier to read, link, and maintain.
