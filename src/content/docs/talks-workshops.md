Technical talks and workshops are a way to turn hard-won implementation lessons into shared practice. The original portfolio grouped this work around video, slides, and teaching formats.

## Themes

- Software testing strategy and automation design.
- Context-driven testing and quality leadership.
- Developer workflows for test infrastructure and tooling.
- Technical education for practitioners moving from concepts to application.

## Teaching

Chris has taught through Association for Software Testing programs, including Black Box Software Testing material. This work emphasizes reasoning, practical judgment, and the ability to select testing approaches based on context rather than ceremony.

## Portfolio Fit

For developer relations, this work shows the ability to:

- Teach technical material to mixed-experience audiences.
- Convert product and engineering concepts into memorable examples.
- Facilitate practitioner conversations where feedback and education flow both ways.
